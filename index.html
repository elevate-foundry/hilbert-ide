System Pipeline

State Space Components

H_I

Intent Space

Holds the raw intent string and its embedding vector. All planning begins here.

H_G

Execution Graph Space

A directed acyclic composition of typed tool operators with explicit data dependencies.

H_A

Artifact Space

The output of execution — a typed dictionary of produced artifacts keyed by name.

H_H

History (Ledger) Space

An append-only, content-addressed ledger of all committed states. Supports full replay.

Quick Start

Clone the repository and run the demo pipeline in seconds:

# Clone git clone https://github.com/elevate-foundry/hilbert-ide.git cd hilbert-ide # Run the full demo pipeline (CLI) python3 cli.py demo # Execute a single step python3 cli.py step "write a file" --context '{"filename":"out.txt","content":"hello"}' # Replay any committed state by its SHA-256 hash python3 cli.py replay <sha256-hash> # View the full append-only ledger python3 cli.py history # Start the REST API server uvicorn api:app --host 0.0.0.0 --port 8765
